1. Cài đặt JDK 16
2. Cài đặt Tomcat phiên bản 9.0
3. Cài đặt MySQL
4. Cài đặt Eclipse

Mở Terminal / Command Line và thực hiện những bước sau đây.


2. Tạo database trên MySql với bằng cách Execute file **data.sql**
